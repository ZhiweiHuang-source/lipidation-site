# High-Throughput Analysis Scripts

This repository contains scripts for the analysis of protein library HPLC, MALDI, and THT data, as well as Fiji macros for confocal image analysis.

## Repository Structure

- **HPLC/**: Scripts for processing High-Performance Liquid Chromatography data.
- **MALDI/**: Scripts for Matrix-Assisted Laser Desorption/Ionization data analysis.
- **THT/**: Scripts for Thioflavin T aggregation assay analysis.
- **ML/**: Scripts for Machine Learning based predictions and plotting.
- **ImageAnalysis/**: Fiji macros for confocal image analysis.
