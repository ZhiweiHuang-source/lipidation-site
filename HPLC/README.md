# HPLC Analysis

Scripts for processing and plotting HPLC data.

## Files

- `plot_hplc.py`: Main script to plot HPLC traces from CSV data.
- `make_hplc_pptx.py`: Generates a PowerPoint presentation from the plots.
- `verify_counts.py`: Verification utility.

## Usage

Ensure your HPLC output CSV files are in place, then run:

```bash
python plot_hplc.py
```
