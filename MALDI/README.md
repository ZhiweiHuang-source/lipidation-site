# MALDI Analysis

Scripts for analyzing MALDI-TOF mass spectrometry data.

## Files

- `plot_maldi_constructs.py`: Plot MALDI spectra for different constructs.
- `plot_maldi_book1.py`: Plotting utility for specific datasets.
- `make_maldi_ppt.py`: Generates a PowerPoint summary of the MALDI plots.
- `si_table3.py`: Generates SI Table 3 data.

## Usage

```bash
python plot_maldi_constructs.py
```
